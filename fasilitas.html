Fasilitas Unggulan Kami

Kami menyediakan lingkungan belajar yang modern dan lengkap untuk mendukung siswa menjadi ahli di bidang Teknik Komputer dan Jaringan.

Laboratorium Jaringan Komputer

Pusat praktik untuk merancang, mengkonfigurasi, dan mengelola jaringan. Siswa belajar langsung dengan perangkat standar industri.

  • Perangkat Router Mikrotik & Cisco
  • Switch Manageable & Unmanageable
  • Praktik Routing, Switching, dan VLAN
  • Simulasi Jaringan dengan GNS3 & Packet Tracer

Laboratorium Server & Cloud

Tempat siswa belajar instalasi, konfigurasi, dan maintenance server fisik maupun virtual (cloud).

  • Rack Server untuk Praktik Langsung
  • Instalasi & Administrasi OS Linux & Windows Server
  • Teknologi Virtualisasi (Proxmox/VMware)
  • Pengenalan Platform Cloud (AWS/Azure)

Workshop IoT & Mikrokontroler

Ruang kreatif untuk eksplorasi Internet of Things (IoT), menggabungkan hardware, software, dan jaringan.

  • Board Arduino, ESP32, dan Raspberry Pi
  • Berbagai Macam Sensor dan Aktuator
  • Pemrograman Dasar Mikrokontroler
  • Proyek Smart Home & Monitoring Jarak Jauh
